About Our Trello Clone

This project is a Trello clone developed with Angular, TailwindCSS, and Dexie. It is a web application designed to be local-first , with the option to sync your boards in future versions. Its goal is to replicate Trelloâ€™s core features while providing users with a smooth and efficient task and project management experience.

The purpose of building this clone is to understand and learn modern Angular, making use of Signals, standalone components, services, and related concepts.

The main features include creating boards, lists, and cards, drag-and-drop functionality, real-time updates, and user authentication. The application is designed to be responsive and intuitive.
